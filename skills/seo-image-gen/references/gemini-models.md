## 分辨率等级

使用 `imageSize` 参数控制输出分辨率。请注意 **大写 K** 要求。

| `imageSize` 值 |像素范围 |模型可用性 |使用案例|
|--------------------|-------------|--------------------|----------|
| `512` |高达 512×512 |所有模型 |草稿、快速迭代、低带宽 |
| `1K`（默认）|高达 1024×1024 |所有模型 |标准网络使用、社交媒体 |
| `2K` |高达 2048×2048 | 3.1 Flash |优质资产，细致做工|
| `4K` |高达 4096×4096 | 3.1 Flash |印刷制作、hero image、最终资产 |

**注释：**
- 实际像素尺寸取决于宽高比（例如，4K at 16:9 = 4096×2304）
- 分辨率越高，消耗的代币越多，成本也越高
- 如果未指定 `imageSize`，则默认为 `1K`

## API 配置

### 端点

```
https://generativelanguage.googleapis.com/v1beta/models/{model-id}:generateContent
```

### 必需参数

```json
{
  "contents": [{"parts": [{"text": "your prompt here"}]}],
  "generationConfig": {
    "responseModalities": ["TEXT", "IMAGE"],
    "imageConfig": {
      "aspectRatio": "16:9",
      "imageSize": "1K"
    }
  }
}
```

### 仅图像输出模式
强制模型仅返回图像（无文本响应）：

```json
{
  "generationConfig": {
    "responseModalities": ["IMAGE"]
  }
}
```

### 思维水平
控制模型在生成之前“思考”的程度。更高的级别可以改善复杂的组合，但会增加延迟：

```json
{
  "generationConfig": {
    "thinkingConfig": {
      "thinkingLevel": "medium"
    }
  }
}
```

级别：`minimal`、`low`、`medium`、`high`

### Google Search grounding
让生成基于真实世界视觉参考。支持网页和图像搜索（3.1 Flash）：

```json
{
  "tools": [{"googleSearch": {}}]
}
```

**提示词模式：** `[Search/source request] + [Analytical task] + [Visual translation]`

示例：“搜索最新的 SpaceX 星舰设计，分析其比例和标记，然后在发射台上生成日落时的逼真图像。”

### 多图像输入
最多可提供 14 张参考图像：
- **10 个对象引用**：用于风格、构图或视觉匹配
- **4 个字符引用**：分配不同的名称以保留跨代的功能

对于角色一致性、风格转移和品牌一致的生成很有用。

## 安全过滤器

Gemini 采用两层安全架构：

1. **输入过滤器**：在生成前拦截包含禁止内容的提示
2. **输出过滤器**：分析生成的图像并阻止不安全的结果

| `finishReason` |意义|可重试吗？ |
|----------------|---------|:----------:|
| `STOP` |生成成功|不适用 |
| `IMAGE_SAFETY` |输出被安全过滤器阻挡|改写提示 |
| `PROHIBITED_CONTENT` |违反内容政策 |否 - 主题已被阻止 |
| `SAFETY` |通用安全拦截|改写提示 |
| `RECITATION` |检测到受版权保护的内容 |改写提示 |

**已知问题：** 众所周知，过滤器过于谨慎。良性提示可能会被阻止。如果发生这种情况，请重新措辞。

## 内容凭证

- **SynthID 水印** 始终嵌入生成的图像中（不可见，机器可读）
- **C2PA 元数据** 包含在专业/付费输出中（可验证的来源链）

## 主要限制
- 没有视频生成（仅图像）
- 没有透明背景（PNG，但总是有背景）
- 文本渲染质量各不相同；保持文本少于 25 个字符以获得最佳效果
- 安全过滤器可能会阻止一些提示（暴力、NSFW、公众人物），已知过于谨慎
- 当前 Agent对话之间的会话上下文重置
- `imageSize` 和思维水平取决于MCP package 版本支持

## 本项目适配边界

- 图片生成、模型调用和后处理都可能依赖外部模型或本地工具；未配置时标注“未配置”。
- 生成商业图片时要记录 prompt、模型、成本、用途、版权/品牌约束和是否需要人工复核。
