# 国际 SEO 文化适应概况

> 本文件是 `seo-agents` 的中文详细参考，已按当前项目能力边界适配。命令名、API 名、Schema 类型、JSON 字段和专用 SEO 术语按需保留英文。

分析多语言网站或生成 hreflang 时加载此参考
实施。概要文件指导文化评估超越技术验证。

原创概念：Chris Muller（Pro Hub 挑战赛）

## DACH 地区（德国、奥地利、瑞士）

|尺寸|指南|
|------------|------------|
|正式|高的。使用“Sie”（正式的你）。预期的专业语气。 |
|幽默|保守的。避免在 CTA 中讽刺和双关语。 |
| CTA 风格 |间接优先。 “Jetzt entdecken”优于“Jetzt kaufen”。 |
|信任信号|认证（TUV、ISO）、“Datenschutz”（隐私）、法律要求的 Impressum。 |
|法律 |印象页强制。 DSGVO (GDPR)。 Widerrufsrecht（撤回权）。 |
|数字格式 | 1.000,00（点表示千位，逗号表示小数）|
|日期格式 |年.月.日 |
|货币 |欧元（德国/奥地利）、瑞士法郎（瑞士）|
|颜色象征意义|蓝色=信任，绿色=生态/自然，红色=谨慎（不紧急）|
|品牌替代|沃尔玛 → MediaMarkt、家得宝 → Hornbach、亚马逊 → 奥托/扎兰多 |
|单词扩展|与英语相比 +25-35%（计划更长的标题和按钮文本）|

## 法语（FR、BE、CA-FR、CH-FR）

|尺寸|指南|
|------------|------------|
|正式|中高。默认为“你”。 “Tu”仅适用于青年/休闲品牌。 |
|幽默|成熟时受到赞赏。避免直率/直接的幽默。 |
| CTA 风格 |措辞优雅。 “Decouvrir nos Solutions”胜过“Achetez maintenant”。 |
|信任信号| “Fabrique en France”（法国制造）、专业认证、媒体提及。 |
|法律 |提及所需的法律。 CNIL（数据保护）。 CGV（术语）。 |
|数字格式 | 1 000,00（千位空格，小数点逗号）|
|日期格式 |日/月/年 |
|货币 |欧元 (FR/BE)、加元 (CA-FR)、瑞士法郎 (CH-FR) |
|颜色象征意义|蓝色=稳定，白色=纯洁/奢华，红色=激情 |
|品牌替代|沃尔玛 → 家乐福、亚马逊 → Fnac/Cdiscount、Target → 勒克莱尔 |
|单词扩展|与英语相比 +15-25% |

## 西班牙裔（西班牙、拉丁美洲）

|尺寸|指南|
|------------|------------|
|正式|因国家/地区而异。西班牙：“Usted”正式。拉丁美洲：好坏参半。 |
|幽默|温暖且有关系。接受自嘲式幽默。 |
| CTA 风格 |温暖、有个性。 “Empieza tu viaje”优于“Comprar ahora”。 |
|信任信号|社区证明、家庭/关系主题、名人代言都很受欢迎。 |
|法律 | LOPD（西班牙数据保护）。每个拉丁美洲国家都有自己的法规。 |
|数字格式 |西班牙：1.000,00 / 拉丁美洲：因国家/地区而异 |
|日期格式 | DD/MM/YYYY（大多数）、DD-MM-YYYY（部分拉丁美洲）|
|货币 |欧元（西班牙）、地区货币（MXN、ARS、COP、CLP、PEN）|
|颜色象征意义|红色=能量/激情，黄色=温暖，蓝色=信任|
|品牌替代|沃尔玛 → Mercadona (ES) / Coppel (MX)、亚马逊 → MercadoLibre |
|单词扩展|与英语相比 +15-25% |

## 日语 (JA)

|尺寸|指南|
|------------|------------|
|正式|非常高。 Keigo（敬语）预计在商业中。 |
|幽默|微妙的。在 B2B 中避免直接幽默。 B2C 中可接受的卡哇伊（可爱）元素。 |
| CTA 风格 |微妙而有礼貌。 “Otoiawase”（询问）直接“立即购买”。 |
|信任信号|公司寿命长、ISO 认证、认可机构的认可。 |
|法律 | APPI（个人信息保护法）。德亭正取引法。 |
|数字格式 | 1,000（逗号表示千，在大多数情况下没有小数）|
|日期格式 | YYYY/MM/DD 或 YYYY 年 MM 月 DD 日 |
|货币 |日元（无小数）|
|颜色象征意义|白色=纯洁，红色=活力/庆典，黑色=正式|
|品牌替代|亚马逊 → 乐天、Google购物 → 雅虎！日本购物 |
|单词缩写 | -10-25% vs 英语（日语更紧凑）|

## 默认配置文件（未列出的语言）

在分析没有上述预构建配置文件的区域设置时：

1. **研究形式规范**：检查语言是否有正式/非正式语域（例如韩语有7个语音级别）
2. **检查文本方向**：LTR 与 RTL（阿拉伯语、希伯来语、波斯语、乌尔都语）
3. **验证数字/日期格式**：使用 CLDR（Unicode 通用区域设置数据存储库）
4. **研究法律要求**：隐私法、商业登记、消费者保护
5. **检查单词扩展比例**：日耳曼/斯拉夫语言扩展，CJK语言收缩
6. **验证货币和付款偏好**：可能需要提及本地付款方式
7. **研究颜色含义**：颜色在全球有不同的文化关联

## 如何在分析中使用

在多语言站点上运行 `/seo hreflang` 时：
1. 确定所有语言版本及其目标市场
2. 加载相关文化概况
3. 检查 CTA、信任信号和法律页面是否符合文化期望
4. 将不匹配标记为“文化适应”结果（中等严重程度）
5. 检查每个语言环境版本中数字/日期/货币格式的一致性

## 本项目适配边界

- 优先使用当前已实现的 `seo-agents` CLI 获取证据；没有 CLI 的能力作为 Agent playbook 或后续扩展处理。
- 不把外部数据源、付费 provider、浏览器渲染或地图 API 写成已执行，除非当前任务确实运行并取得证据。
- 每个 finding 必须有可复核 evidence；数据缺失时明确写“未配置”或“缺少输入”。
